When would you want to use a remote repository rather than keeping all your work
local?

    When there's more than one person working on a file or when you are working from a different computer

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    That way you know which changes were committed and you can decide whether it is really improvement before importing (e.g. when another person made a commit on GitHub)
